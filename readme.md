# Setting up

To update nixos with the latest confuguration run:

```sh
sudo nixos-rebuild switch --flake ~/.config/kp/nixos/#default
```

## Solaar setup

- set `Ket/Button Diversion` for `Mouse Gesture Button` to `Mouse Gestures`
- create rule with:
  - key `Mouse Gestures Button`
  - Key press `Super_L`
